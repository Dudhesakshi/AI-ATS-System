# AI-ATS-System 🚀

AI-powered Applicant Tracking System built using Spring Boot, React, MySQL, and PDF processing for intelligent resume analysis, ATS scoring, skill detection, and candidate management.

## 📌 Overview

AI-ATS-System is a full-stack recruitment platform designed to streamline the hiring process by automating resume screening and candidate evaluation.

The system parses uploaded resumes, extracts candidate information, analyzes skills, calculates ATS scores, and helps recruiters manage candidates efficiently.

## ✨ Features

* 📄 Resume Upload and Parsing
* 🤖 AI-based ATS Score Generation
* 🧠 Skill Detection and Analysis
* 👤 Candidate Profile Management
* 🔍 Resume Screening
* 📊 Dashboard for Candidate Tracking
* 📱 Responsive User Interface
* 🔗 REST API Integration
* 💾 Database Storage using MySQL
